Tags /meta tags

Tag São etiquetas que não aparecem na tela do computador que serve para orientar . É uma linguagem para que o navegador entenda.

Meta Tags

Meta Tag ou meta *description* - usamos para descrever o conteúdo de uma página Web. Essas descrições aparecem nos mecanismos de buscas ex: Gloogle, Yahoo…
Dentro de uma Tag coloca- se um

atributo, podemos ter várias tags meta sendo que cada uma terá um atributo.
_meta *name*="description" *content*="*Minha* *primeira* *página* *só* *com* *HTML”_ colocando essa tag meta description, o Google(ferramenta de busca) conseguiria localizar a página.

_meta charset= “UTF-8”_

Identifica o formato de codificação da página. Existe vários formatos diferentes para cada tipo de página idiomas e caractere.
👉🏻 meta charset= UTF-8 - representa um padrão, com ele representa qualquer caractere de qualquer idioma.
👉🏻meta “Viewport”-Fornece ao navegador instruções de como controlar as dimensões e a escala da página.
